# System Generator

A collection of bash scripts designed to automate the creation of a custom Arch Linux installation ISO and the subsequent installation process, including partitioning, encryption, and postinstallation configuration. It aims to streamline the setup of a customized Arch Linux system based on author's personal preferences and requirements.

## Usage

1. Clone: `git clone git@github.com:shashotoNur/sysgen.git`
2. Navigate into the directory: `cd sysgen`
3. Execute: `sudo bash main.sh [phase]`
